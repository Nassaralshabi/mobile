🛠️ خطوات تثبيت Flutter

1 تحميل Flutter SDK:
اذهب إلى flutter.dev وحمل الإصدار الأحدث
2 استخراج الملفات:
استخرج ملف ZIP إلى C:\flutter
3 إعداد PATH:
أضف C:\flutter\bin إلى متغيرات البيئة PATH
4 تشغيل التطبيق:
افتح PowerShell في مجلد mobile وشغل flutter pub get ثم flutter run

📥 ملفات مساعدة

تم إنشاء ملفات batch لتسهيل عملية التثبيت والتشغيل:

📋 setup_flutter.bat ▶️ run_flutter.bat
